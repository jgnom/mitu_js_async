**Практическая работа по JS.**

Для запуска приложения необходимо установить NODE.JS можно скачать по ссылке: https://nodejs.org/en/download/prebuilt-installer
Так же после установки нужно инициализировать проект командой npm init.
Для работы нужен бодуль ws установить его командой npm install ws

Запуск: node script.js
